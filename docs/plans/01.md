# Big Bang Theory Mode — "Before the Stars"

## Context

CosmicX currently boots straight into the Solar System Sandbox. The user wants a second, separate experience: a timeline-driven journey from the Big Bang to the far future — mode selection screen, scrubbable epoch timeline, cinematic opening animation, spacetime expansion (not objects flying outward), particle→atom→star→galaxy→solar-system progression, and a documentary-style camera. Simplified-but-convincing visuals are explicitly allowed. The sandbox must be untouched, and the minimalist "liquid glass" design language preserved.

**Decision (user-confirmed):** Big Bang mode lives in a new self-contained `bigbang.html` beside `index.html`. Rationale: index.html is ~5,020 lines and `tests/simulator.test.mjs` slices it with first-match regex section markers (`section()`, `functionSource()`) — growing it ~2,500 lines risks marker drift and name collisions. Mode select fades to black (`#fade`) and navigates.

## Hard constraints (verified in tests/simulator.test.mjs)

- Test `requests fullscreen from Start…` (lines 200–213) extracts `function setupTitleScreen() {` → `const clock = new THREE.Clock();` and asserts the fullscreen block's content/ordering. **Do not modify the existing `#startBtn` handler body**; append the new `#bigbangBtn` listener inside that same region.
- Test at line 124: `border-radius` must appear **exactly 13 times** in index.html's `<style>`. New mode-card CSS must use square corners (matches design language anyway).
- Test at line 89 rejects a syntactically invalid module; keep index edits minimal.

## 2. bigbang.html — structure

Head: same font links + importmap (three@0.170.0 jsdelivr). `<style>`: **first rule** `html,body{margin:0;height:100%;overflow:hidden;background:#000}` (prevents white flash on navigation), then copied tokens/components (~200 lines): `:root` vars verbatim, `[hidden]`, `.glass`, `.btn`, `.chip`, `.section-label`, `.slider-row` + full range-input block, `#fade`, `#cinebars`, toast styles, reduced-motion block, plus new `#bbTitle/#bbBar/#bbPanel/#epochCard` rules.

Overlays: `#bbTitle` (title-card clone: h1 "Before the Stars", tagline = subtitle, `#beginBtn`, top-left `#backLink` "← Modes" anchor to `index.html`), `#fade`, `#cinebars`, `#toasts`, `#epochCard`, `#bbPanel`, `#bbBar`.

Script sections (banner comments like index):
1. **CONFIG + QUALITY** — bloom params, particle-count budget, pixelRatio clamp (index:1327 pattern).
2. **Copied utilities** (verbatim, self-contained): `hash2/valueNoise/fbm` (index:1063–1088), `createPointSpriteTexture` (1195), `createGlowTexture` (1209), `createNebulaTexture` (1223), `createPlanetTexture` (1104), `makeRockGeometry` (1959), `accretionDiskMaterial` (2825), `paintRangeFill` (4129), `toast` (~3599), `BloomClampShader` (1288). Skip `LensingShader` — `// ponytail: lensing pass skipped, add ShaderPass(LensingShader) before OutputPass if BH shot feels flat`.
3. **Renderer/composer**: RenderPass → BloomClamp → UnrealBloom → OutputPass → FXAA (index:1356–1371 minus lensing). ACES tone mapping. `bloomPass` mutable — epoch env drives strength per frame.
4. **EPOCHS table + timeline core** (§3).
5. **Subsystems** (§4) — each `build*()` returns `{ group, apply(u, env), tick(dt) }`, pushed to `systems[]`. Scene graph: `universeGroup` (scaled by a(u) — the expansion) + `fixedGroup` (background starfield).
6. **`applyEpoch(u)`** — single state function: blended env → every `system.apply(u, env)`, bloom strength, bg color, `universeGroup.scale`, epoch card text, scrubber + `paintRangeFill`, cinematic camera pose. **Pure in `u`** — no `Date.now`/`performance.now`/`Math.random` (test-enforced); dt drives only cosmetic shimmer (`uTime` uniforms).
7. **Transport**: `stepTimeline(dt)`, `setBBSpeed(i)`, `setDirection(d)`, `jumpToEpoch(i)`.
8. **Camera** (§5).
9. **`setupBBUI()` / `setupBBTitle()`** — Begin click: fullscreen request with same `.catch(() => toast('Fullscreen unavailable — continuing in this window'))` pattern → hide overlay → `T.playing = true` → reveal bar/panel.
10. **Loop + debug handle**:

```js
function update(dt) { stepTimeline(dt); for (const s of systems) s.tick(dt); applyEpoch(T.u); }
window.bang = { THREE, camera, controls, EPOCHS, T, systems, renderer,
  step: dt => update(dt), render: () => composer.render(),
  setU: u => { T.u = THREE.MathUtils.clamp(u, 0, 1); applyEpoch(T.u); },
  jumpToEpoch, epochAt, get bloomPass() { return bloomPass; } };
```

## 3. Timeline data model

11 epochs with authored screen-time weights (pure log-time starves the late epochs):

```js
const EPOCHS = [
  { id:'singularity', label:'Singularity',        timeLabel:'t = 0',              tempLabel:'∞',       span:4, env:{…}, cam:{from,to}, desc:'…' },
  { id:'planck',      label:'Planck Epoch',       timeLabel:'10⁻⁴³ s',            tempLabel:'10³² K',  span:6, … },
  { id:'inflation',   label:'Inflation',          timeLabel:'10⁻³⁶–10⁻³² s',      tempLabel:'10²⁸ K',  span:7, … },
  { id:'particles',   label:'Particle Formation', timeLabel:'10⁻⁶ s – 3 min',     tempLabel:'10¹³ K',  span:11, … },
  { id:'atoms',       label:'Atoms Form',         timeLabel:'380,000 yr',         tempLabel:'3,000 K', span:9, … },
  { id:'firststars',  label:'First Stars',        timeLabel:'200 Myr',            span:12, … },
  { id:'galaxies',    label:'First Galaxies',     timeLabel:'1 Gyr',              span:12, … },
  { id:'milkyway',    label:'Milky Way Forms',    timeLabel:'~8 Gyr ago',         span:9, … },
  { id:'solar',       label:'Solar System Forms', timeLabel:'4.6 Gyr ago',        span:13, … },
  { id:'present',     label:'Present Day',        timeLabel:'13.8 Gyr',           tempLabel:'2.7 K',   span:8, … },
  { id:'future',      label:'Future Universe',    timeLabel:'10¹⁴ yr and beyond', span:9, … },
];
```

- Helpers: `normalizeEpochs()` (spans → `u0/u1` once at load), `epochAt(u)`, `epochProgress(u)`, `uForEpoch(i)`.
- `env` per epoch: scalar targets `{ scale, bloom, bg, gridA, foamA, plasmaA, plasmaHeat, particleMix, cmbA, starMix, galaxyMix, spiralMix, solarMix, futureMix, redshift }`. `envAt(u)` smoothstep-blends current→next epoch over the last ~20% of each span — this one function makes every transition seamless in both directions. Subsystems consume env scalars only; they never know epochs.
- Playback: `const T = { u:0, speedIdx:2, dir:1, playing:false, cinematic:true }`, `BB_SPEEDS = [0.25,0.5,1,2,4,8]`, `BASE_RATE = 1/90` (full journey ≈90 s at 1×). `stepTimeline`: `T.u = clamp(T.u + dt*T.dir*speed*BASE_RATE, 0, 1)`, auto-pause at ends.
- `jumpToEpoch(i)`: adjacent → 0.8 s quintic tween of `T.u`; far → `#fade` dip, snap, fade off (documentary hard cut).
- **Scrub-reversibility rule**: no dt-spawned persistent state anywhere. All beats (star ignitions, collision flashes) derive from hashed functions of `u`. This is why no interaction-log/rebuild machinery is needed, unlike the sandbox.

## 4. Visual subsystems (Three.js primitives already proven in this codebase)

| System | Technique | Env driver |
|---|---|---|
| `buildQuantumFoam` | ~4k `Points`, point-sprite tex, additive; per-point phase attr, vertex-shader flicker via `uTime`; tiny shell at origin | `foamA` |
| Big Bang flash | one glow `Sprite`, HDR color (e.g. `Color(3,3,2.8)`), scale/opacity pulse as `u` crosses inflation `u0`; env bloom peaks 3–4× | `bloom` |
| `buildSpaceGrid` | `LineSegments` 12³ lattice, additive ~0.10 opacity, inside `universeGroup` → expands with a(u) automatically; fades after atoms | `gridA` |
| `buildPlasma` | ~20k `Points` ball; blackbody color-ramp uniform: heat 1→0 = blue-white→orange→deep red; size shrinks as it cools | `plasmaA`, `plasmaHeat` |
| `buildParticles` | one ~15k `Points` cloud with **two position attributes** `aScatter`/`aBound` + `uMix` uniform: `mix(aScatter, aBound, uMix)` in vertex shader — quark soup condenses to protons/neutrons/hydrogen; zero per-frame CPU, perfectly reversible | `particleMix` |
| CMB shell | `BackSide` sphere with `createNebulaTexture` mottling, orange→near-black tint; plasma fades out as it fades in ("universe turns transparent") | `cmbA` |
| `buildFirstStars` | ~40 units {nebula sprite + glow sprite}; per-unit hashed phase staggers births: cloud shrinks (collapse) → glow pops (ignition; bloom = fusion flash) | `starMix` |
| `buildGalaxies` | ~60 imposters merged in ONE BufferGeometry (log-spiral point distributions, per-galaxy center attr) + core glow sprites; positions **comoving** in `universeGroup` → Hubble's law by construction (world pos = comoving × scale). Redshift: fragment shader lerps color toward red by `min(1, dist × uRedshift)` | `galaxyMix`, `redshift` |
| `buildMilkyWay` | hero spiral: 3-layer `Points` (bulge/arms/halo ~12k) + core glow + small Sgr A* sphere with `accretionDiskMaterial`; irregular clumps → grand spiral via uMix morph | `spiralMix` |
| `buildSolarForm` | nebula sprites lerp ball→disk plane; sun glow grows→ignites; ~200 `InstancedMesh` planetesimals (`makeRockGeometry`) orbiting in-plane; 4–5 protoplanet spheres (`createPlanetTexture` rocky) growing with progress; collision flashes at hash-deterministic positions keyed on `floor(progress*12)` | `solarMix` |
| Present day | finished mini system: sun + planets on faint orbit rings + 2-layer parallax starfield (index starLayers pattern) | `solarMix`=1 fade-in |
| Future | stars redden/dim, recession accelerates (extra scale growth), starfield fades, accretion disk last bright object, near-black end; `u=1` auto-pauses | `futureMix` |

**Expansion model**: `a(u)` authored via `env.scale` keys (≈0.01 → inflation ×40 jump → ramp to 1.0 at present → 1.6 future), blended by `envAt`. **`#expansionRate` slider** (0.2–3.0, default 1) multiplies the post-inflation portion of the scale and the `uRedshift` uniform — faster recession + deeper reddening, live.

**Perf budget**: ≈55k total points high / ≈22k low quality (halve counts + pixelRatio 1); one draw call per subsystem; morphing via uniforms, never per-frame attribute writes.

## 5. Cinematic camera

- Per-epoch `cam:{from:{pos,look,fov}, to:{...}}`; pose = blend(from, to, eased epochProgress) + small Lissajous drift overlay (miniature of the title drift, index:4849–4886). Author `epoch[N].to ≈ epoch[N+1].from` for continuous handoffs; deliberate hard cuts (galaxies→milkyway interior) use a 0.3 s `#fade` dip.
- `#cinebars` letterbox on while `T.cinematic`.
- Free camera: `#bbCamBtn` (+ Esc back) toggles `T.cinematic`; free mode enables OrbitControls (target = Milky Way core during formation epochs — satisfies "rotate camera around galaxies while forming"). Re-entering cinematic: `glideTo(current→keyframe, 1.2 s)` tween, no snap.
- Camera pose is a function of `u` in cinematic mode → scrubbing scrubs the documentary; reverse plays the film backwards.

## 10. Risks

1. **Bloom + fill rate** dominates cost → pixelRatio clamp, ≤55k points, merged geometries, uniform morphing only.
2. **CDN offline** → blank page; same exposure as index.html (all textures procedural). Accepted.
3. **index.html test regex fragility** → all edits inside the `setupTitleScreen`…`clock` window, startBtn handler untouched, run suite right after P0.
4. **Reversibility trap** → hard rule: no dt-spawned state; hashed-from-u beats only.
5. **Fullscreen lost on navigation** → re-requested by Begin click.
6. **White flash on navigate** → `background:#000` first rule in bigbang.html.
