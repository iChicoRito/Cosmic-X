# Objective

## Redesign the Main Menu and Enhance the "Before the Stars" Experience

---

## Role

You are a game UI/UX designer and gameplay systems developer. Approach this objective with the mindset, vocabulary, and priorities of designing polished user interfaces, cinematic experiences, and visually cohesive game systems.

---

## Description

Redesign the main menu to provide a more engaging and polished introduction to the game by introducing a dedicated mode selection flow with interactive previews. Improve the visual presentation of the interface through better alignment, modern glassmorphism styling, and smoother transitions between menu states. Enhance the "Before the Stars" mode by significantly upgrading its visual effects, particle systems, cinematic camera movement, and overall presentation while maintaining consistency with existing assets used in the sandbox experience. The final result should deliver a smoother, more immersive, and visually cohesive experience across both the menu and gameplay.

---

## Primary Objective

Redesign the game's menu flow and substantially improve the presentation, visuals, and cinematic quality of the "Before the Stars" mode while maintaining consistency with existing game assets.

---

## Secondary Objectives

* Redesign the title screen into a two-stage menu with a dedicated mode selection interface.
* Add interactive hover previews that dynamically transition between available game modes.
* Apply a modern glassmorphism style to menu buttons.
* Improve the alignment and presentation of menu typography.
* Enhance the visual quality of the Big Bang sequence with richer effects, particles, and lighting.
* Improve the smoothness and cinematic quality of camera transitions.
* Replace placeholder celestial bodies with the same Solar System models used in Sandbox mode.
* Expand the galaxy formation sequence to include additional galaxies beyond the Milky Way.

---

## Success Criteria

* The initial title screen displays only the **Cosmic X** title and a centered **Start** button.
* Selecting **Start** transitions to a separate mode selection screen.
* Hovering over **Become the Creator** previews the existing Solar System environment and updates the title accordingly.
* Hovering over **Before the Stars** previews the Big Bang environment and updates the title accordingly.
* Menu buttons use a subtle glassmorphism design.
* The title and subtitle remain centered throughout all menu states.
* Camera transitions are smooth and cinematic.
* The "Before the Stars" mode includes noticeably richer visual effects, particles, and environmental detail.
* Planet formation uses the same Solar System models as Sandbox mode.
* The galaxy formation sequence includes the Milky Way, Andromeda, and Messier galaxies.

---

## Context & Dependencies

* The current title screen displays the **Cosmic X** title with a **Start** button.
* The existing Solar System environment used for the **Become the Creator** preview already exists and should be reused.
* The Solar System models currently used in Sandbox mode should also be used during planetary formation in the "Before the Stars" mode to maintain visual consistency.
* The current "Before the Stars" mode already contains a Milky Way sequence that should be expanded rather than replaced.

## Detailed Breakdown

### Main Menu Flow

Maintain the existing landing screen displaying the **Cosmic X** title and **Start** button. Selecting **Start** should transition to a separate mode selection interface rather than immediately launching gameplay. The mode selection screen should provide interactive previews that dynamically update the background and displayed title based on the currently hovered option.

### Interactive Mode Previews

Hovering over **Become the Creator** should transition the background to the current Solar System environment while updating the displayed title accordingly. Hovering over **Before the Stars** should instead transition the background to the existing Big Bang environment while updating the title to match the selected mode. Both previews should transition smoothly without abrupt visual changes.

### User Interface

Modernize the interface by applying a subtle glassmorphism style to menu buttons while keeping the overall design clean and polished. Ensure the title and subtitle remain perfectly centered across every menu state for a consistent presentation.

### Before the Stars Enhancements

Improve the overall presentation of the Big Bang sequence through richer visual effects, more impressive particle systems, smoother transitions, and higher overall visual quality. The experience should feel significantly more cinematic while preserving the original gameplay intent.

### Camera Improvements

Refine camera movement throughout the sequence to eliminate abrupt transitions and improve cinematic flow. Camera motion should remain smooth and visually cohesive during every stage of the experience.

### Planet Formation

When the Solar System begins forming, use the same planetary models currently employed in Sandbox mode so both game modes maintain a consistent visual style.

### Galaxy Formation

Expand the existing galaxy formation sequence beyond the Milky Way by incorporating both the Andromeda galaxy and additional Messier galaxies while preserving the current progression of the sequence.
