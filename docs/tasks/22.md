# Objective
## Introduce the Wormhole Galaxy Simulation

---

## Description

Introduce the Wormhole Galaxy as a new, standalone fictional galaxy within the existing **As the Gods Will** galaxy simulations. Unlike the existing galaxies, this environment should revolve around a massive wormhole that serves as the central interactive element while still allowing players to freely create, place, and simulate celestial bodies just as they can in the other galaxy simulations. The Wormhole Galaxy should provide its own unique gameplay through customizable wormhole physics, dynamic gravitational effects, teleportation mechanics, and immersive visual effects, creating a distinct cosmic experience while preserving the familiar sandbox simulation workflow. The implementation must integrate seamlessly into the existing galaxy selection system without modifying or affecting the behavior, mechanics, or performance of the current galaxy simulations.

---

## Primary Objective

Develop and integrate the Wormhole Galaxy as a new fictional sandbox galaxy that supports the same core simulation capabilities as the existing galaxies while introducing a massive wormhole with customizable physics, dynamic gravitational effects, teleportation mechanics, and immersive visual effects.

---

## Success Criteria

- A new **Wormhole Galaxy** is available alongside the existing galaxies.
- Players can freely simulate celestial bodies within the Wormhole Galaxy using the same core mechanics available in the other galaxy simulations.
- The massive wormhole supports customizable physics, dynamic gravitational effects, teleportation mechanics, and immersive visual effects.
- The Wormhole Galaxy functions as an independent simulation environment without impacting the functionality, performance, or behavior of existing galaxies.

---

## Constraints

- Preserve all existing galaxy simulations without modifying their mechanics or behavior.
- Maintain consistency with the existing simulation controls and user experience.

---

## Context & Dependencies

- The feature extends the existing **As the Gods Will** galaxy simulation mode.
- The Wormhole Galaxy should follow the same sandbox simulation principles as the current galaxies while introducing its own unique wormhole-focused mechanics.
- Integration should fit naturally into the existing galaxy selection and simulation workflow.

---

## Supporting Tasks

### Galaxy Integration
- Add the Wormhole Galaxy as a new selectable galaxy within the existing galaxy menu.
- Ensure it initializes independently from the other galaxy simulations.

### Core Simulation
- Support the same simulation capabilities available in the existing galaxies, including creating, placing, moving, and interacting with celestial bodies.
- Preserve the existing controls and simulation experience for consistency.

### Wormhole Mechanics
- Implement a massive central wormhole as the galaxy's defining feature.
- Support customizable wormhole physics.
- Add dynamic gravitational effects that influence nearby celestial bodies.
- Implement teleportation mechanics through the wormhole.
- Include immersive visual effects that enhance the wormhole without compromising performance.

### Compatibility
- Ensure the new galaxy operates independently and does not interfere with the logic, physics, assets, or performance of any existing galaxy simulation.
